## Requirements

-   npm - [Link](https://www.npmjs.com/get-npm)
-   Expo - [Link](https://expo.io/)
    -   Make sure you have Expo CLI installed: `npm install -g expo-cli@6.3.2`

## How To Run:

**Install Dependencies**

-   Navigate to the project folder and run: `npm install`

**Run Expo App**

-   `expo start`
